📟 Termux (Terminal para Android):
• Disponível no F-DROID
• Versão recomendada: 0.118.0 ou superior
• Função: Ambiente de execução do script

===============================================================
2. COMANDOS DE INSTALAÇÃO E EXECUÇÃO
===============================================================

🔧 ETAPA 1: Instalação do Termux
• Baixe e instale o F-DROID
• No F-DROID, pesquise por "Termux" e instale

🔧 ETAPA 2: Preparação do Termux
Abra o Termux e execute:

pkg update
pkg upgrade -y
pkg install python -y
pkg install git -y
pkg install wget -y

🔧 ETAPA 3: Download do Script
git clone https://github.com/AdrianoBduz/Bitcoin.git
cd Bitcoin

🔧 ETAPA 4: Permissões de Execução
chmod +x install.sh
chmod +x bitcoin_finder.py

🔧 ETAPA 5: Instalação das Dependências
./install.sh

🔧 ETAPA 6: Preparar Arquivos Necessários
• Crie/copie o arquivo wordlist.txt com 2048 palavras BIP39
• Crie/copie o arquivo P2PKH.txt com endereços Bitcoin

🔧 ETAPA 7: Executar o Script
python bitcoin_finder.py

===============================================================
3. ARQUIVOS NECESSÁRIOS
===============================================================

📄 wordlist.txt:
• Lista de 2048 palavras em inglês do padrão BIP39
• Formato: Uma palavra por linha
• Exemplo: 
  abandon
  ability
  able
  about
  above
  ...

📄 P2PKH.txt:
• Lista de endereços Bitcoin existentes (formato P2PKH)
• Formato: Um endereço por linha
• Exemplo:
  1A1zP1eP5QGefi2DMPTfTL5SLmv7DivfNa
  1BitcoinEaterAddressDontSendf59kuE
  ...

===============================================================
6. SOLUÇÃO DE PROBLEMAS
===============================================================

❌ Erro: "Arquivo não encontrado"
• Verifique se wordlist.txt e P2PKH.txt estão no diretório

❌ Erro: "Permission denied"
• Execute: chmod +x bitcoin_finder.py

❌ Erro: "Python not found"
• Execute: pkg install python -y

❌ Script muito lento:
• Feche outros aplicativos
• Conecte o carregador
• Use termux-wake-lock
